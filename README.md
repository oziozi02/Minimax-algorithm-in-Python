# Minimax-algorithm-in-Python
This repository contains minimax algorithms implemented in Python<br/>
for a game created using the Pygame library.<br/>
More details about the algorithms and the game can be found in the .pdf [file](IS_DZ2_2026.pdf).
### Algorithms
- MaxN
- Minimax
- Minimax with Alpha-Beta pruning

Implementations of the algorithms(agents) can be found in [agents.py](isdz2/agents.py).
### Requirements
- Pygame library(Can be installed via pip using ``` pip install pygame ```).
